# windgate-pigments
For the projootian

- Module to model mixing colours in Defaultio's Windgate (with a model of the ingame pigment ID system)
- Additionally allows for conversion between Brickcolour, float RGB (0-1), RGB (0-255), and recipe objects

- Tested to be around 85% accuracy, please check generated recipes before use


# Dependencies
Ensure that you have pymixbox (https://github.com/scrtwpns/mixbox/tree/master)
